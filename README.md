Pre-requisites
----------------
- xCode 12.4 or higher
- MAC machine
- Review [Swift Basics](https://docs.swift.org/swift-book/LanguageGuide/TheBasics.html) and [Swift Closures](https://docs.swift.org/swift-book/LanguageGuide/Closures.html)
